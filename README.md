# Date and Time Picker Project

A React-based application that provides a custom interface for selecting a date from a calendar and a specific time.

# Project Overview

This project was built to practice handling date and time logic in React without using external UI libraries. It features two main components:

* Date Selection: A calendar grid for choosing a day, month, and year.

* Time Selection: A modal for picking hours, minutes, and the period (AM/PM).

# Features

* Custom-built calendar logic.

* State management for date and time synchronization.

* Reset and Submit functionality.

# How to Run Locally

1. Clone the repository to your machine.

2. Run npm install to download dependencies.

3. Run npm start to open the app in your browser.
